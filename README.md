# Particle-Simulator

JavaScript canvas particle simulator.

Basically a 2D gravity simulator, in the browser.

## Building

You must have `npm`, `node`, and `gulp` already installed.

```
npm install
gulp
```

## Running

Open `path/to/Particle-Simulator/index.html` in your web browser.
